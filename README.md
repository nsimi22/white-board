# WhiteBoard

A lightweight, standalone whiteboard app built with vanilla HTML, CSS, and Canvas API — no dependencies, no build step.

## Features

| Tool | Shortcut | Description |
|------|----------|-------------|
| Pen | `P` | Freehand drawing |
| Eraser | `E` | Erase pixels |
| Rectangle | `R` | Draw rectangles |
| Ellipse | `O` | Draw ellipses / circles |
| Line | `L` | Draw straight lines |
| Text | `T` | Click to place text |

- Stroke color, fill color, fill toggle
- Adjustable brush size (1–60 px) and opacity
- Undo / Redo (`Ctrl+Z` / `Ctrl+Y`)
- Clear canvas
- Save as PNG

## Usage

Open `index.html` in any modern browser — nothing to install.
